# IF - Else
En Go la sintaxis y la sintaxis son parecidos a otros lenguajes

``` go
if x > 10 {
    return "X mayor a 10"
}
```

Ahora la diferencia que puede tener es que puede crear variables con alcanze limitado es decir, varibles en la misma declaracion que el if

``` go
if y:=10,y >5{
    return "Hola Y"
}

```

## Else
Igual a otros lenguajes la secuencia que se activa cuando no es verdadera la condicion del IF


``` go
if y:=10,y >5{
    return "Hola Y"
}else{
    return "Y no es mayor a 5"
}

```

## if else

Funcionamiento igual que otros lenguajes es concatenacion donde agregamos otro bloque de condicion

``` go
if y:=10,y >5{
    return "Hola Y"
}else if y>0{
    return "Y es mayor a 0 y menor o igual a 5"
}else{
    return "Y es negativo"
}

```
