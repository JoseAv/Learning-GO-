# Variables

Forma de guardar a un pedazo de memoria para guardar algun valor en el.

## Var
Var es la forma para declarar una variable en go, podemos indicarle el tipo que es y puede tambien no asignarse a nada

``` go
func main(){
	var x int = 5 // int el tipo -> se lo indicamos
    var y = 4 // -> infiere que es tipo int por la asignacion

    var z,v = 3,4 // -> Creando varias variables

    var value // -> no se inicializo

	fmt.Println(x)
}

```

## Short declaration

En Go se puede utlizar una forma corta para asignar valores, necesita si o si ser asignada a un valor

``` go
func main(){
    y := 5 // infiere que es int

}

```


## Var vs :=
Si bien puede parecer lo mismo, tienen algunas diferencias a tomar en cuanta para elegir una o otro

***Diferencias***

***Var:*** 

1. Puede usarse en cualquier lugar, dentro y fuera de la funcion
2. Puede declararse sin un valor
3. Si quieres especificar el tipo, util para objetos mas complejos

***:= :***
1. Solo funciona dentro de una funcion
2. Necesita ser asignada inmediatamente
3. Infiere el tipo 



## Var global
Igualmente que las funciones que necesitan empezar por mayuscula para poder importarse, esto si declaramos esto afuera de un paquete

``` go

var value = 5 // Esta no se exporta
var Position = 10 // Esta si se exporta

```

para poder usarla, usamos con el nombre del paquete que importamos, seria paquete.Position como si importaramos una funcion normal

## Const

Las constantes son valores fijos que nunca cambien, tambien puede ser exportados de la misma forma que var

``` Go
const DataBase = "localhost:5432" // se puede importar
const user = "admin"                // no se puede importar

const (
    StatusOk = 200
    statusError = 500
) // Forma de declarar varias constantes

```


Las constantes solo pueden contener

- Numeros
- Strings
- Booleanos

Cualquier otra cosa no podremos usar const, si queremos usar noPrimitivos, usamos funciones normales
