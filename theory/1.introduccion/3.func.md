# Funciones

En Go hay diferentes tipos de funciones y aqui nombrare algunas y para que sirve

## Named function

es la funcion normal de toda la vida

``` go
func sayHi(name string){
    return fmt.Println(name," dice Hola");
}

```

## Funcion Anonima

Funcion que se puede guardar en una variable o es autoInvocatia.

``` go
var saveFunction = func(){return "HI"}

func(message string){
    fmt.Println(message);
}("Hola como estan")
```


## Funcion Variadica

Funcion que acepta multiples argumentes

```go
func countNumbers (numeros ...int) int {
    total := 0
    for _,num := range numeros{
        total +=nums

    }
    return total
}


```

## Retornar mas de un error

En Go las funciones pueden retornar mas de un valor 

``` go
import ("errors")

func dividir(x,y int)(int, error){
    if y<=0 {
        return 0,errors.New("No puede ser zero")
    }

    // nill: no hubo error
    return x/y, nill

}


func main(){
    division, err := dividir(10,20)
    if err!= nil{
        fmt.Println("Hubo un error")
    }

    fmt.Println("El resultado es",division)
}

```
