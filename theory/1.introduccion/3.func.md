# Funciones

En Go hay diferentes tipos de funciones y aqui nombrare algunas y para que sirve

## Named function

es la funcion normal de toda la vida

``` go
func sayHi(name string){
    return fmt.Println(name," dice Hola");
}

```

## Funcion Anonima

Funcion que se puede guardar en una variable o es autoInvocatia.

``` go
var saveFunction = func(){return "HI"}

func(message string){
    fmt.Println(message);
}("Hola como estan")
```


## Funcion Variadica

Funcion que acepta multiples argumentes

```go
func countNumbers (numeros ...int) int {
    total := 0
    for _,num := range numeros{
        total +=nums

    }
    return total
}


```

## Funciones como tipos

```go
type operation func(int,int)int

func anyOperation(a,b int, op operation) int {
    return op(a,b)
}

func sumar(x, y int) int {
	return x + y
}

func restar(x, y int) int {
	return x - y
}

func multiplicar(x, y int) int {
	return x * y
}
```
