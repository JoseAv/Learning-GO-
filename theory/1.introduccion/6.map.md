## Map con range

Cuando nosotros iteramos un map este map no va a salir de aleatoria.

1. No hay orden garantizado
2. Es alateatoro por intencio
3. Implementacion con un hash table

para poder enterder porque los maps salen de forma aleatoria tendremos que enteder estos conceptos

- Has Table.
- Bucket y sus partes

### Hash table

Hash table es una estructura que funciona con Buckets de 8 posiciones.

##### Bucket  
Bucket es una coleccion ordenada que puede guardar 8 claves y valores guardaras en indices.

Sus partes son

- TopHash: Es una forma ingeniosa de convertir la clave en los 8 bits mas altos y ayuda a comprar rapidamente los valores por valores en bits.
- Key: es el punto de entrada al valor.
- Value: valor guardado como la definicion de una palabra en el diccionario.
- Overflow: Nos indica si ya esta lleno el bucket y nos manda a otro.

### Concluion de Map con Range

Ahora que ya desglozamos podemos no entender que tiene que ver con que Map cuando lo iteramos salga de forma random

- ***Antes:*** En versiones anteriores a Go si salian del mismo orden, ya que los buckets si estan ordenados por indices y empezaban por el 0 hasta el ultimo bucker.

- ***Actualmente:***  En versiones actuales cambiaron eso ya que habian desarrolladores que tenian la mala practica de depender de este orden, el proceso actual es el siguiente, eligen un bucket al hacer por 

ejemplo

``` go
// Tenemos 8 buckets
    Bucket 1[0-7]
    Bucket 2[0-7]
    Bucket 3[0-7]
    Bucket 4[0-7]


// Elige uno al azar

    Bucket 3[0-7]

// entonces ahora los recorrera asi 

    Bucket 3[0-7]
    Bucket 4[0-7]
    Bucket 1[0-7]
    Bucket 2[0-7]


// si lo volvieramos a recorrer puede ser que ahora lo haga en otro orden

    Bucket 2[0-7]
    Bucket 3[0-7]
    Bucket 4[0-7]
    Bucket 1[0-7]

```

## keys

Hasta ahora solo hemos usado keys de tipo string para asignarla como clave, pero las keys pueden ser de estos tipos

 
***Basicos:***
- String
- Float64
- Bool
- String

***Compuestos:***
- Array -> map[[3]int]string
- Struc sin slices/maps -> map[struc{x,y,int}]string
- Punteros: map[*int]string
- Canales: map[chan int]string

***No permitidos:***
- Slices
- Maps como clabes
- Struc con slices/maps
