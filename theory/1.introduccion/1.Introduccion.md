## Package  Â¿Que es?

Los Package son una forma que tiene go similar a las clases de indicar que codigo es del mismo paquete, es muy util para agrupar funciones en grupos.

### Main
Go al ser un lenguaje compilado necesita un main para indicar su punto de entrada, el lenguaje para ejecutarse siempre va a buscar el paquete main y la funcion main.

Puede ser cualquier archivo con cualquier nombre pero siempre necesita el paquete main junto con la funcion o no funcionara, ya que no podria encontrar el compilador su punto de partida.

### Import

Cuando quieres el codigo de un package lo importas, asi puedes utlizar sus funciones

- Mayusculas: Quieres que la funcion se importe
- Minisculas: Quieres la funcion privada

#### Import a carpetas
Hay una convencion en Go que el nombre de la carpeta coincida con el nombre del paquete si nosotros queremos importar algun paquete para usar sus funciones tendremos que utilizar el nombre de nuestro go.mod/nombre_carpeta luego con esto ya podemos utilizar las utilidades que nos trae ese paquete 


``` go
my "basic-golan/mymath" // paque renombrado usamos my y no mymath
```

``` go
"basic-golan/mymath" // paquete y carpeta coincidin usamos mymath
```


``` go
"basic-golan/mymath" // si carpeta y paquete no coinciden usamos el nombre del paquete, pero si el paquete comparte nombre con otro paquete entonces tendremos error y lo renombrara a nombre de la carpeta para evitar errores
```

quedaria esto ->
``` go
mymath "basic-golan/mymath"
```

## Comando Go

Existen varios comando por CLI utiles con GO, estos son solo alguno ve a la documentacion para saber mas

https://go-dev.translate.goog/doc/cmd?_x_tr_sl=en&_x_tr_tl=es&_x_tr_hl=es&_x_tr_pto=tc

- go run : Compila y corre el codigo, no genera .exe
- go build: compila y crea el ejecutable
- go mod init: crea un proyecto o modulo similar al package json de js
- go get: Descar las libreria como npm
- go install: instala herramientas globalmente
- go mod dowload: descarga todo lo que hay en el init
- go test: ejecuta las pruebas
- go fmt: Formate codigo
- go doc: Mustra documancion de paquetes
- go mod tidy: Limpia dependencias no usadas y agrega las faltantes
