# For

En Go la unica forma de loop es For es bastante versatil ya que puede funcionar como un while, for of, o un for tradicional.

## For con indices

``` go

    func main(){
        for i:=0 ; i<5; i++{
            fmt.Println(i)
        }
    }
```


## Slice vs Array
en otros lenguaje no existe el termino slice en js los arrays son todos por iguales y con un valor fijo son tuplas pero en go

- Slice: Array de valor variable
- Array: Array con valor fijo

Sintaxis
``` go 
// Slice
number := []int{1,2,3}

// Array
number := [2]int{1,2}
```


## For range

La palabra reservada range nos permite iterar sobre un slice range lo que hace es devolverte el indice y el valor, cuando esta dentro del for itera el slice

***Tip :***
Range solo puede usarse bajo el contexto de for y nada mas.

``` go
for i,value := range []int{1,2,3}{
    fmt.Println(value,i)
}

```


## While

En Go no existe el while pero hay una forma de trabajar con la misma forma

``` go
condition := true
for condition{

    break;
}

```

## Do while

En Go no existe while ni do while pero puedes hacer que funcione el loop con ese mismo comportamiento

``` go
for {

    // Operaciones a hacer

    if(!condition){
        break
    }

}

```

O


``` go
condition := true
for ok:=true;ok;ok = condition{
    // Codigo

    // ❌ Esto no aqui creas una variablle
    condition:= false

    // ✅ Ahora no
    condition = false
}

```
