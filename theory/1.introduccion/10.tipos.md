# Tipos de datos

En Go existen diferentes tipos de datos.

Primitivos:

- ***bool*** | True/False | Flag o condicionales

- ***String*** | Cadena de caracteres | Representa texto |

- ***int, int8, int16, int32, int64*** | Entero | int por defecto|16/32/64 para controlar el tamaño de los enteros |

- ***uin, uin16, uint32, uin64*** | Entero sin signo | cuando no queremos negativos |

- ***float32, float64*** | Valores numericos reales | numeros con puntos | 32/64 -> por el sistema |


- ***byte*** === uint8 | 8 bites es un byte| trabajar con datos binarios |

- ***rune*** === int32 | cuando queremos representar un solo caracter que ocupa mas de un byte |❌ -> emojis ocupa 2 string |

- ***complex64, complex128*** | representa parte real y parte imaginaria | N + iN |

# Valores por defecto


- ***bool*** | False |
 
- ***String*** | "" |

- ***int, int8, int16, int32, int64*** | 0 |

- ***uin, uin16, uint32, uin64*** | 0 |

- ***float32, float64*** | 0 |


- ***byte*** | 0 |

- ***rune*** | 0 |

- ***complex64, complex128*** |0 + i0 |
