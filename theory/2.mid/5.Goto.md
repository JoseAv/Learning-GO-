# Goto

Goto salta a la etiqueta End, y todo el codigo que podia quedar pasa a no ser leido

``` go
func main(){

    fmt.Println("#1")
        goto End
    fmt.Println("No me ejecutare")

End:
    fmt.Println("Salida")


}
```

Esto no es recomemdable y la verdad es mejor dejar de terminar y despues retornar.
